Moonlander

MoonLander is a simple game written in C using the SDL2 library.
The game simulates a lunar lander that needs to safely land on the moon's surface. The terrain is randomly generated every time and the gravitational pull and fuel levels are determined by the level chosen by the player. A higher difficulty means more extreme terrain with fever places to land, higher gravity, and less fuel.

Features:

  • Randomly generated terrain
  • Multiple levels, with increasing difficulty
  • High scores

Gazdálkodj Okosan

This is a C# implementation of the popular Hungarian board game "Gazdálkodj Okosan".
The application tracks the players' bank balance as in newer iterations of the game the players not only use cash but also have a bank account where they can keep their money and even receive interest.
The project is a work in progress. The back-end is working, although it lacks many features like validation.

Camo Tracker

CamoTracker is a Java-based desktop application that helps track weapon camo challenges in
Call of Duty: Modern Warfare II (2022).
The application has been developed using the Swing framework to provide an intuitive and user-friendly interface. The application allows gamers to track their progress towards completing weapon challenges and monitor their mastery challenges.

Features:

  • Weapon Challenge Tracking: The application allows users to track their progress towards completing weapon challenges.
  • Mastery Challenge Tracking: CamoTracker also enables users to track their progress towards completing mastery challenges. Mastery challenges are more complex challenges that require users to complete all previous challenges to achieve a higher level of mastery.
  • JSON Persistence: CamoTracker uses JSON format to persist data, enabling users to save and load their progress with ease.

Requires Java SE Development Kit (JDK) version 17 or above.
Works on all modern PC operating systems.

Music Player

This is a Java-based music player application that allows you to play .wav files. The application is developed using only core Java and comes with a user-friendly interface using Swing.

Features:

  • Plays .wav files
  • Simple and easy-to-use user interface
  • Play, pause, and stop functionality
  • Ability to seek through the song
  • Displays the current playing song's title
  • Repeat functionality

Planned features:

  • More hotkeys
  • Repeat song a set amount of times
  • Dark mode
  • Listen together on local network or online

Requires Java SE Development Kit (JDK) version 11 or above.
Works on all modern PC operating systems.
